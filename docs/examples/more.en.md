# More examples

Additional "Hello, World!" samples in other languages. Use the tabs to switch.

=== "Rust"
    ``` rust
    fn main() {
        println!("Hello, World!");
    }
    ```

=== "PHP"
    ``` php
    <?php
    echo "Hello, World!";
    ```

=== "Swift"
    ``` swift
    print("Hello, World!")
    ```

=== "Kotlin"
    ``` kotlin
    fun main() {
        println("Hello, World!")
    }
    ```

=== "C"
    ``` c
    #include <stdio.h>

    int main(void) {
        printf("Hello, World!\n");
        return 0;
    }
    ```

=== "Bash"
    ``` bash
    echo "Hello, World!"
    ```
